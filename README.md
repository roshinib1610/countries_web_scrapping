# countries_web_scrapping

This project is a Python-based web scraping script that extracts country information such as name, capital, population, and area from a website and stores the data in an Excel file.

**Features**
Scrapes country data from a live website
Extracts:
Country Name
Capital
Population
Area (km²)
Saves data into a structured Excel file (countries.xlsx)
Simple and beginner-friendly implementation

**Technologies Used**
Python
BeautifulSoup (for parsing HTML)
Requests (for HTTP requests)
OpenPyXL (for Excel file handling)

**Output**

The Excel file will contain the following columns:

S.No
Country
Capital
Population
Area (km²)

**Use Cases**
Learning web scraping basics
Data extraction and automation
Practice working with Excel files in Python
